Main visualization function.

Writes time stamped VTK datasets for visualization.

VTK Double Array

vtkDoubleArray type pcoords is filled with the data in meshPoints.

vtkPoints type outputPoints is filled with the data in pcoords.

Connectivity is written to vtkCellArray type outputCells

Scalar property

Previously collected data which are outputPoints, outputCells, scalarProperty, are written to vtkPolyData type polydata var.

Whatever collected in polydata above is written to the vtk file below. vtkDataSetWriter is for leagacy VTK format, vtkXMLDataSetWriter is for VTK XML format.

vtkXMLDataSetWriter *writer = vtkXMLDataSetWriter::New();
