Detailed Description

This class defines ELEMENT LEVEL DATA STRUCTURE.

This data structure holds the variables determined for an element. An element have a connectivity (the node numbers of the element) and face groups which indicates which face has which boundary condition type. Other than these you will need to define element mass stiffness matrices or any other matrices that will be necessary during the solution stage.
