Detailed Description

This class defines LINEAR TRIANGULAR MASTER ELEMENT.

In the mesh, there are elements with different sizes and shapes. Instead of integrating the equations on each element, we use a reference element, which simplifies the integration. Note that the reference element is same for all the elements in the mesh.
